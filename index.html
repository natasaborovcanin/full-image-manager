Full Image Manager - OpenCart extension

*** A must to have kind of extension ***

Full Image Manager is an OpenCart extension based on elFinder. Very powerful options and real user friendly interface. Try our online demo and find out why is Full Image Manager a "must have" kind of extension exactly right for you

Features :

  • All operations with images and folders on a remote server (copy, move, upload, create folder, rename, etc.)
  • High performance server beckend and light client UI
  • Background file upload with Drag & Drop HTML5 support
  • List and Icons view
  • Standard methods of file/group selection using mouse or keyboard
  • Move/Copy files with Drag & Drop
  • Archives create/extract (zip, rar, 7z, tar, gzip, bzip2)
  • Quicklook, preview for common file types
  • Edit images
  • Calculate directory sizes
  • Thumbnails for image files

  • And much more..
